Function for reading one item from the buffer.

After reading item from the buffer, HeadPtr will be incremented and count field will be decremented. If buffer is empty HeadPtr cant be incremented.

CB_StatusMessage_t CB_put ( circular_buffer_t *  cb,
void *  item 
)

Function for adding one item to the buffer.

After pushing item to the buffer count will be incremented for one and tailPtr will be incremented for objectSize. If buffer is full the oldest item will be overwritten, if headPtr is at the oldest item in the moment of overwritting headPtr will be incremmented for and count field wont.

Parameters
[in]cbPointer to the circular buffer to which you want to add item
[in]itempointer to the Item you want add to the buffer
Returns
a status message of function – MS_OK or MS_ERROR
