Circular buffer

Api for CB which can hold any type of object
First step in using this API is to initiate circular buffer using init_CB, which will return status massage if CB is successfully created or not.

Another two functions of importance are CB_put() and CB_get() function, which are used for pushing item in buffer and reading item from it.

At the beginning the pointer value of field headPtr and tailPtr will be same, as the items are inserted and read over time, the pointers will increase.

If the buffer is full and there are more items to push in it, the oldest items will be overwritten.
